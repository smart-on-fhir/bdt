Class Console

Each Test object has an instance of this class attached to it. Tests use that to add entries to the console which ends up being a valuable information for test reporters (as well as for humans)

byTags

  • byTags(tags: string | string[]): ConsoleEntry[]
    • Defined in src/lib/Console.ts:190

    Returns a subset of the console entries that match the given argument. If the argument is a string, it should be comma-separated or space-separated list of tags. In this case only entries having all of the listed tags are returned. To get entries having any of the listed tags pass an array instead Example:

    console.byTags("a, b") // a AND b
console.byTags(["a", "b"]) // a OR b

    Parameters

    • tags: string | string[]

    Returns ConsoleEntry[]
